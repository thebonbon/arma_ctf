# arma_ctf
Arma 3 capture the flag PvE mission generator

## Installtion
Execute the init.sqf

## Usage
Spawn the flag with the following command:
Arguments:
 * 0: Defend Radius <NUM>
 * 1: Capture Speed <NUM> (Slow 100 | Normal 10 | Fast 1 )

```sqf
[50, 10] call BON_fnc_SpawnFlag
```
